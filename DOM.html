la manipolazione del DOM in JavaScript:

cos'è il DOM?

il DOM ovvere document object model è tutto ciò che si trova in html,
quindi sostanzialmente quello che appare visibile nello schermo dato che html è la struttura del programma.

cos'è la manipolazione del DOM?

JavaScript consente di gestire,creare,eliminare e modificare tutti gli oggetti di html per esempio di modificare le coordinate
di un elemento div attraverso il CSS(la parte grafica del codice).
es.//html
div id="p1" style="width: 40px;height: 40px;background-color: black;position: absolute;left: 0px;top: 0px;"/div
//script
const p1 = document.getElementById("p1");
let x = 0;
let y = 0;
function move(){ p1.style.top= y + 2 +"px";
p1.style.left= x + 2 +"px";
}
addEventListener("click",()=>{
move();
});

come si crea un elemento?

per creare un elemento in html da javascript usiamo la formula
document.createElement()
andando a inserire nelle parentesi il tipo di oggetto es.("div")

come si rimuove un elemento?

per rimuovere un elemento bisogna dichiararlo all'interno di una variabile e poi usare il metodo .remove() per rimuoverlo
es.const esempio = document.getElementById("esempio");
esempio.remove();

come si gestisce un elemento?

è possibile associare un elemento all'interno di un altro con il metodo .appendchild()
es.document.body.appendChild(nuovoDiv);
 in questo esempio l'elemento div viene associato al body.
caso opposto,è possibile rimuovere un oggetto già associato a un'altro attraverso .removechild()
es.let contenitore = document.getElementById("contenitore"); let elementoDaRimuovere = document.getElementById("figlio"); contenitore.removeChild(elementoDaRimuovere);
