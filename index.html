JavaScript

benvenuto nella guida per JavaScript scritta da Angelo Del Piano,iniziamo subito

introduzione:

JavaScript è uno dei pilastri portanti di html,css,js ed è la parte dinamica del tutto
poichè consente di gestire eventi e di manipolare la struttura del dom(document object model),ovvere ciò che esiste in html.
js è molto usato nella programmazione di siti web e giochi per la sua versatilità ma passiamo alla parte pratica.

le variabili:

la prima cosa che c'è da imparare di js sono le variabili.
quasi in ogni linguaggio di programmazione esistono le variabili,scatole di dati che servono al programma ad avere delle informazioni,
in js esistono 3 tipi di variabili:
1.let:la variabile più usata poichè è molto dinamica.
2.var:vecchia variabile,attualmente non si usa quasi più grazie a let
3.const:variabile constante che non cambia dopo la prima assegnazione.

le funzioni:

js dà la possibilità di raggruppare più istruzioni di codice in una sola,attraverso le funzioni.
sostanzialmente le funzioni sono pacchetti di codice personalizzabili in maniera semplice.
sono una delle cose più utili di js e rendono più facile la scrittura del programma.
es. function saluta(){console.log("ciao");};
saluta()// output:ciao

la manipolazione del DOM:

come abbiamo detto prima JavaScript consente di manipolare il contenuto di html cioè il DOM,
permettendo un'ampia dinamica del programma. quest'argomente è abbastanza complicato se non si conosce html
per tanto è consigliato approfondire prima html.
JavaScript permette di creare elementi nel html e di modificarli a poprio piacimento.

gli eventi:

JavaScript consente di fare controlli in base all'evento desiderato,per esempio
se l'utente clicca il tastro sinistro del mouse su un oggetto del html JavaScript ascolta quell'evento.
es.addeventlistener("click",()={
console.log("hai cliccato");
}); // quando l'utente clicca il tasto sinistro del mouse, output:hai cliccato
