le funzioni in JavaScript:

in javascript le funzioni sono dei blocchi di codice che semplificano la scrittura del programma

in javascript esistono 8 tipi di funzioni e sono:

funzioni dichiarate:

le funzioni dichiarate sono definite con la keyword function e possono essere usate
prima della loro dichiarazione grazie all'hoisting.
function somma(a, b) {
return a + b;
}
console.log(somma(2, 3)); // Output: 5

le funzioni espresse:

le funzioni espresse sono assegnate a una variabile e non subiscono l'hoisting.
const moltiplica = function(a, b) {
return a * b;
};
console.log(moltiplica(4, 5)); // Output: 20

le arrow function:

le arrow function(o funzioni freccia) sono una variabile più compatta delle funzioni e
sono ottime per scrivere funzioni anonime(cioè senza nome) brevi.
const sottrai = (a, b) => a - b;
console.log(sottrai(10, 3)); // Output: 7
 se c'è un solo parametro si possono omettere le parentesi const doppio = x => x * 2;
console.log(doppio(4)); // Output: 8

le funzioni IIFE:

le funzioni IIFE(Immediately Invoked Function Expression) sono delle funzioni che si auto eseguono non appena vengono definite
(function() {
console.log("Eseguito subito!");
})();
// Output: "Eseguito subito!"

le funzioni costruttore:

le funzioni costruttore sono funzioni che permettono di creare funzioni dinamicamente e sono introdotte da new function
const sommaDinamica = new Function("a", "b", "return a + b");
console.log(sommaDinamica(2, 3)); // Output: 5

le funzioni ricorsive:

le funzioni ricorsive sono funzioni che si chiamano da sole, usate per problemi come il fattoriale.es.
function fattoriale(n) {
return n === 0 ? 1 : n * fattoriale(n - 1);
}
console.log(fattoriale(5)); // Output: 120

le funzioni di callback:

le funzioni di callback sono funzioni passate come argomento ad altre funzioni.
es.function saluta(nome, callback) {
console.log("Ciao " + nome);
callback();
}
function mostraMessaggio() {
console.log("Benvenuto!");
}
saluta("Marco", mostraMessaggio);
// Output:
// Ciao Marco
// Benvenuto!

le funzioni Asincrone:

le funzioni Asincrone sono usate per operazioni asincrone come chiamate API.
es.async function fetchData() {
let response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
let data = await response.json();
console.log(data);
}
fetchData();
